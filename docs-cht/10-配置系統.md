# 配置系統

## 概述

OpenClaw 使用 JSON5 格式的配置檔案，支援環境變數替換、包含指令、Zod Schema 驗證等功能。配置採用嚴格驗證——未知鍵會導致 Gateway 拒絕啟動。

## 配置檔案位置

### 預設路徑

```
~/.openclaw/openclaw.json
```

### 自訂路徑

```bash
# 環境變數
export CLAWDBOT_CONFIG_PATH=/path/to/config.json5

# 命令列參數
openclaw --config /path/to/config.json5
```

### 配置檔案（Profile）

```bash
# 使用 dev 檔案
export CLAWDBOT_PROFILE=dev
# 會載入 ~/.openclaw/openclaw.dev.json

# 或命令列
openclaw --profile dev
```

## 配置結構

### 完整配置範例

```json5
{
  // 模型配置
  "models": {
    "default": "claude-sonnet",
    "entries": {
      "claude-sonnet": {
        "provider": "anthropic",
        "model": "claude-3-5-sonnet-latest"
      },
      "gpt-4o": {
        "provider": "openai",
        "model": "gpt-4o"
      },
      "local": {
        "provider": "ollama",
        "model": "llama3.2"
      }
    }
  },
  
  // 提供者配置
  "providers": {
    "anthropic": {
      "apiKey": "${ANTHROPIC_API_KEY}"
    },
    "openai": {
      "apiKey": "${OPENAI_API_KEY}"
    },
    "ollama": {
      "baseUrl": "http://localhost:11434"
    }
  },
  
  // Agent 配置
  "agents": {
    "default": {
      "id": "default",
      "name": "OpenClaw",
      "model": "claude-sonnet",
      "workspace": "~/Projects",
      "identity": {
        "name": "OpenClaw",
        "soul": "You are a helpful AI assistant."
      },
      "tools": {
        "exec": { "enabled": true },
        "browser": { "enabled": true }
      }
    }
  },
  
  // 頻道配置
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "${TELEGRAM_BOT_TOKEN}",
      "allowlist": ["@username"],
      "streaming": { /* 串流預覽設定 */ }
    },
    "whatsapp": {
      "enabled": true,
      "allowlist": ["+1234567890"]
    },
    "discord": {
      "enabled": true,
      "token": "${DISCORD_BOT_TOKEN}",
      "streaming": { /* 串流預覽設定 */ }
    }
  },
  
  // 插件配置
  "plugins": {
    "entries": {
      "matrix": {
        "enabled": true,
        "config": {
          "homeserver": "https://matrix.org",
          "userId": "@bot:matrix.org"
        }
      }
    }
  },
  
  // Gateway 配置
  "gateway": {
    "mode": "local",
    "port": 18789,
    "bind": "loopback",
    "token": "${GATEWAY_TOKEN}",
    "reload": {
      "mode": "auto"  // 熱重載支援
    }
  },
  
  // 路由配置
  "routing": {
    "rules": [],
    "default": "default"
  },
  
  // 會話設定
  "session": {
    "dmScope": "per-channel-peer"
  },
  
  // 其他設定
  "timezone": "Asia/Taipei",
  "locale": "zh-TW"
}
```

## 配置區塊詳解

### models - 模型配置

```json5
{
  "models": {
    "default": "claude-sonnet",
    "entries": {
      "claude-sonnet": {
        "provider": "anthropic",
        "model": "claude-3-5-sonnet-latest",
        "maxTokens": 8192,
        "temperature": 0.7
      }
    }
  }
}
```

### providers - 提供者配置

```json5
{
  "providers": {
    "anthropic": {
      "apiKey": "${ANTHROPIC_API_KEY}"
    },
    "openai": {
      "apiKey": "${OPENAI_API_KEY}"
    },
    "mistral": {
      "apiKey": "${MISTRAL_API_KEY}"
    },
    "ollama": {
      "baseUrl": "http://localhost:11434"
    },
    "openrouter": {
      "apiKey": "${OPENROUTER_API_KEY}"
    },
    "bedrock": {
      "region": "us-east-1",
      "accessKeyId": "${AWS_ACCESS_KEY_ID}",
      "secretAccessKey": "${AWS_SECRET_ACCESS_KEY}"
    }
  }
}
```

### agents - Agent 配置

```json5
{
  "agents": {
    "default": {
      "id": "default",
      "name": "OpenClaw",
      "model": "claude-sonnet",
      "workspace": "~/Projects",
      "identity": {
        "name": "OpenClaw",
        "soul": "You are a helpful AI assistant.",
        "boot": "Always respond in Traditional Chinese.",
        "user": "User: Developer"
      },
      "tools": {
        "exec": {
          "enabled": true,
          "allowedCommands": ["ls", "cat", "grep"],
          "deniedCommands": ["rm -rf", "sudo"]
        },
        "browser": { "enabled": true, "headless": true },
        "node": { "enabled": true }
      },
      "hooks": {
        "beforeReply": "hooks/before-reply.js",
        "afterReply": "hooks/after-reply.js"
      }
    }
  }
}
```

### channels - 頻道配置

各頻道的串流預覽統一使用 `channels.<channel>.streaming` 路徑配置。

### gateway - Gateway 配置

```json5
{
  "gateway": {
    "mode": "local",
    "port": 18789,
    "bind": "loopback",
    "token": "${GATEWAY_TOKEN}",
    "tls": {
      "enabled": false,
      "cert": "/path/to/cert.pem",
      "key": "/path/to/key.pem"
    },
    "cors": {
      "enabled": true,
      "origins": ["http://localhost:*"]
    },
    "reload": {
      "mode": "auto"
    }
  }
}
```

## 特殊功能

### 環境變數替換

```json5
{
  "providers": {
    "anthropic": {
      "apiKey": "${ANTHROPIC_API_KEY}"
    }
  }
}
```

### $include 指令

```json5
{
  "$include": ["./providers.json5", "./agents.json5"],
  "models": { "default": "claude-sonnet" }
}
```

### 註解

JSON5 支援 JavaScript 風格的註解。

## 配置載入流程

```
1. 解析候選路徑
   - 環境變數 CLAWDBOT_CONFIG_PATH
   - Profile 檔案 ~/.openclaw/openclaw.{profile}.json
   - 預設檔案 ~/.openclaw/openclaw.json
       ↓
2. 讀取 JSON5 檔案
       ↓
3. 解析 $include 指令（遞迴載入）
       ↓
4. 環境變數替換（${VAR}）
       ↓
5. Zod Schema 驗證（未知鍵拒絕啟動）
       ↓
6. 應用預設值
       ↓
7. 路徑標準化
```

## CLI 配置管理

### 查看配置

```bash
# 列出所有配置
openclaw config list

# 取得特定配置（敏感值自動脫敏）
openclaw config get models.default
openclaw config get channels.telegram.enabled
```

### 設定配置

```bash
openclaw config set models.default gpt-4o
openclaw config set channels.telegram.enabled true
```

### 編輯配置

```bash
openclaw config edit
```

### 重新載入

```bash
openclaw config reload
```

## 新增配置選項

### session.maintenance - 會話自動維護

```json5
{
  session: {
    maintenance: {
      mode: "warn",         // "enforce" 或 "warn"
      pruneAfter: "30d",
      maxEntries: 500,
      rotateBytes: 10485760 // 10MB
    }
  }
}
```

### commands.allowFrom - 命令授權分離

```json5
{
  commands: {
    allowFrom: {
      default: ["*"],
      discord: ["user_id_1", "user_id_2"],
      whatsapp: ["+1234567890"]
    }
  }
}
```

### cron.sessionRetention - Cron 會話保留期

```json5
{
  cron: {
    sessionRetention: "24h"  // 設為 false 停用清理
  }
}
```

### tools.byProvider - 按提供商限制工具

```json5
{
  tools: {
    byProvider: {
      "anthropic": ["exec", "read", "write", "browser"],
      "ollama": ["read", "write"]
    }
  }
}
```

### 工具失敗回覆

工具失敗回覆預設隱藏詳細錯誤，需 `/verbose on` 或 `/verbose full` 顯示。

## 安全建議

1. **不要在配置中硬編碼敏感資訊** - 使用環境變數 `${VAR}`
2. **限制配置檔案權限** - `chmod 600 ~/.openclaw/openclaw.json`
3. **定期輪換 Token 和金鑰**
4. **`config get` 自動脫敏** - 敏感值在輸出中被遮蔽
