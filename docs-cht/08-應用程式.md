# 原生應用程式

## 概述

OpenClaw 提供 iOS、Android、macOS 三個原生應用程式，都透過 WebSocket 連接 Gateway 伺服器進行通訊。

## 應用程式架構

```
                    ┌─────────────────────────────────────┐
                    │            Gateway                  │
                    │         (WebSocket Server)          │
                    └─────────────────────────────────────┘
                         ▲              ▲              ▲
                         │              │              │
              WebSocket  │   WebSocket  │   WebSocket  │
                         │              │              │
               ┌─────────┴───┐   ┌──────┴──────┐   ┌───┴─────────┐
               │   macOS     │   │    iOS      │   │   Android   │
               │    App      │   │    App      │   │     App     │
               │             │   │             │   │             │
               │ - Menu Bar  │   │ - Node Mode │   │ - Node Mode │
               │ - Local GW  │   │ - Camera    │   │ - Camera    │
               │ - Chat UI   │   │ - Voice     │   │ - SMS       │
               │ - Canvas    │   │ - Location  │   │ - Screen    │
               └─────────────┘   │ - Watch App │   └─────────────┘
                                 └─────────────┘
```

## iOS 應用

### 目錄結構

```
apps/ios/
├── project.yml              # XcodeGen 專案定義
├── Sources/                 # 原始碼
│   ├── Info.plist           # 應用程式資訊
│   ├── Gateway/             # Gateway 連接
│   ├── Onboarding/          # 設定流程
│   ├── Screen/              # 螢幕功能
│   ├── Voice/               # 語音功能
│   ├── Services/            # 服務
│   └── EventKit/            # 事件套件
├── WatchApp/                # Apple Watch 應用
├── WatchExtension/          # Watch 擴充
└── Tests/                   # 測試
```

### 主要功能

| 功能 | 說明 |
|------|------|
| Node Mode | 作為節點連接 Gateway，提供裝置能力 |
| Camera | 拍照並傳送給 AI |
| Voice Wake | 語音喚醒功能 |
| Location | 提供位置資訊 |
| Canvas | WebView 顯示 A2UI |
| Watch App | Apple Watch 整合 |
| Setup Code | 配對碼功能 |

### 狀態管理

```swift
// 使用 Observation 框架（推薦）
@Observable
class NodeAppModel {
    var connectionState: ConnectionState = .disconnected
    var gatewayUrl: String?
    var capabilities: NodeCapabilities
    
    func connect() async { /* 連接 Gateway */ }
    func handleInvoke(_ request: NodeInvokeRequest) async -> NodeInvokeResult {
        /* 處理節點呼叫 */
    }
}
```

### 建置與執行

```bash
# 生成 Xcode 專案
cd apps/ios && xcodegen generate

# 開啟專案
open OpenClaw.xcodeproj

# 建置
pnpm ios:build

# 執行
pnpm ios:run
```

**注意**: 優先使用已連接的真實裝置（iOS + Android），再使用模擬器。


### 主要功能

| 功能 | 說明 |
|------|------|
| Menu Bar | 選單列常駐應用 |

### 建置與執行

```bash
# 打包應用程式
pnpm mac:package

# 開啟應用程式
pnpm mac:open

# 重啟（開發用）
pnpm mac:restart
# 或使用腳本
scripts/restart-mac.sh
```

**注意**: 不要透過 SSH 重新建置 macOS 應用；必須直接在 Mac 上執行。

### Gateway 管理

macOS 應用內建 Gateway 管理。Gateway 目前僅作為選單列應用執行，沒有獨立的 LaunchAgent。
- 透過 OpenClaw Mac app 啟動/停止
- 驗證/查看進程：`launchctl print gui/$UID | grep openclaw`

## 開發指南

### 協議同步

```bash
# 生成 TypeScript 協議
pnpm protocol:gen

# 生成 Swift 模型
pnpm protocol:gen:swift
```

### SwiftUI 最佳實踐

```swift
// 使用 Observation 框架（推薦）
@Observable
class MyModel {
    var value: String = ""
}

// 不要使用 ObservableObject（除非相容性需要）
// class MyModel: ObservableObject { ... }
```

觸碰相關程式碼時，應將現有 `ObservableObject` 遷移至 `@Observable`。

### 版本位置

| 平台 | 檔案 |
|------|------|
| CLI | `package.json` (version) |
| Android | `apps/android/app/build.gradle.kts` (versionName/versionCode) |
| iOS | `apps/ios/Sources/Info.plist` (CFBundleShortVersionString/CFBundleVersion) |
| macOS | `apps/macos/Sources/OpenClaw/Resources/Info.plist` |

### 測試

```bash
# iOS 測試
cd apps/ios && xcodebuild test ...

# Android 測試
pnpm android:test

# macOS 測試
cd apps/macos && swift test
```

**重啟應用** = 重新編譯/安裝並重新啟動（不只是 kill/launch）。
