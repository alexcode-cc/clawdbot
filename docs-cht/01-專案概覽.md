# Moltbot 專案概覽

## 什麼是 Moltbot？

Moltbot 是一個強大的 AI 助手平台，支援多種訊息通道（WhatsApp、Telegram、Discord、Slack、Signal、iMessage 等），讓使用者能夠透過熟悉的聊天介面與 AI 互動。

### 核心特色

1. **多通道支援** - 連接超過 20 種訊息平台
2. **AI Agent** - 基於 Claude、GPT 等大型語言模型的智慧代理
3. **工具整合** - 程式碼執行、瀏覽器控制、檔案操作等
4. **跨平台** - 支援 macOS、iOS、Android、Linux、Windows
5. **可擴充** - 完整的插件系統

## 版本資訊

- **目前版本**: `2026.1.27-beta.1`
- **Node.js 需求**: `>=22.12.0`
- **套件管理器**: `pnpm@10.23.0`

## 主要功能

### 1. 通訊頻道
- **內建頻道**: WhatsApp、Telegram、Discord、Slack、Signal、iMessage
- **擴充頻道**: Matrix、MS Teams、Zalo、LINE、Mattermost、Nostr、Twitch 等

### 2. AI 功能
- **對話式 AI**: 自然語言互動
- **程式碼助手**: 程式碼生成、除錯、解釋
- **工具呼叫**: 檔案操作、網頁瀏覽、Shell 執行
- **多模態**: 圖片、音訊、文件理解

### 3. Gateway 伺服器
- **WebSocket RPC**: 即時雙向通訊
- **多客戶端**: Web UI、CLI、行動應用同時連接
- **會話管理**: 對話歷史、上下文壓縮

### 4. 原生應用
- **macOS 選單列應用**: Gateway 管理、聊天介面
- **iOS 應用**: 行動節點、相機、語音
- **Android 應用**: 行動節點、SMS、螢幕錄製

## 專案歷史

Moltbot 源自對「隨處可用的 AI 助手」願景的追求。專案最初聚焦於 WhatsApp 整合（使用 Baileys 協議），後逐步擴展到多通道支援，並發展出完整的 Agent 框架和插件生態系。
