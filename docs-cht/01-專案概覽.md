# OpenClaw 專案概覽

## 什麼是 OpenClaw？

OpenClaw 是一個強大的多頻道 AI 閘道器平台，支援多種訊息通道（WhatsApp、Telegram、Discord、Slack、Signal、iMessage 等），讓使用者能夠透過熟悉的聊天介面與 AI 互動。

### 核心特色

1. **多通道支援** - 連接超過 35 種訊息平台
2. **AI Agent** - 基於 Claude、GPT、Mistral 等大型語言模型的智慧代理
3. **工具整合** - 程式碼執行、瀏覽器控制、檔案操作、音樂/影片生成等
4. **跨平台** - 支援 macOS、iOS、Android、Linux、Windows
5. **可擴充** - 完整的插件系統（50+ 個擴充套件）
6. **Fast Mode** - Anthropic / OpenAI 快速模式切換
7. **背景任務 (TaskFlow)** - SQLite-backed 背景執行控制平面（原 ClawFlow，已更名）
8. **MCP 遠端支援** - SSE/HTTP transport、tool namespacing、channel bridge
9. **Dreaming 記憶系統** - 多階段記憶促進（sleep phases、weighted recall、REM preview）
10. **Prompt Cache 穩定性** - 正式的 cache 穩定性規則與診斷工具
11. **多媒體生成** - 音樂生成（ComfyUI）、影片生成（Runway/xAI/Alibaba/Qwen）

- **技能/插件安全掃描器**: 偵測危險執行、加密挖礦、程式碼混淆等

- **設備配對管理**: `device-pair` 擴充套件

- **執行審批**: exec 工具安全策略
- **沙箱繼承**: `sessions_spawn` 沙箱邊界不允許降級
- **TOCTOU 防護**: 工作區安全寫入防護 symlink 競爭條件

- **Prompt Spoofing 防護**: 中和入站訊息中的偽造 `[System Message]` 標記
